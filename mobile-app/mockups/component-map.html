01
Architecture Layers

Six horizontal layers from UI at top to platform at bottom. Each layer has a single responsibility and a defined interface to adjacent layers. No layer skips — the UI never calls XrayBridge directly.

The single allowed cross-layer skip: XrayBridge.ts (Service Layer) wraps the native module directly. All other services call through XrayBridge.ts — no other file imports NativeModules.

Any state can transition to error on XrayBridge failure.
clearError() returns to idle.

The ProtocolManager selects the protocol before XrayBridge.startXray() is called. If the connection enters error, the AIRouteSelector evaluates the fallback chain and may automatically retry with a different protocol or server — recorded as an entry in aiStore.decisionLog.
